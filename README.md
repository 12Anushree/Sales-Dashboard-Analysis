## 📌 Project Overview

An interactive **Sales Dashboard Analysis** project that explores sales data to uncover trends, top-performing products, regional performance, and revenue insights.  

## 🎯 Objectives

- Analyze overall sales performance across regions, categories, and time periods
- Identify top-selling products and most profitable categories
- Discover seasonal trends and monthly/quarterly revenue patterns
- Build interactive visualizations for stakeholder reporting
- Provide actionable business recommendations

## 📂 Dataset

- **Source:** [Kaggle Datasets](https://www.kaggle.com/datasets)
- **Records:** 10,000+ transactions
- **Features:** Order ID, Product, Category, Region, Sales, Profit, Quantity, Date, Customer Segment

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Pandas | Data manipulation & cleaning |
| Matplotlib | Static visualizations |
| Microsoft excel | Interactive dashboards |
| Microsoft Excel | Development environment |
